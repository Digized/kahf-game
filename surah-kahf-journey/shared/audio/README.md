# Ambient Audio (Optional)

This directory is for nature sounds (halal):
- cave-drips.mp3
- wind-gentle.mp3
- ocean-waves.mp3
- garden-birds.mp3
- river-flow.mp3
- desert-wind.mp3
- rain-gentle.mp3

These are optional enhancements. The experience works without them.

Free sources:
- Freesound.org (CC0 licensed)
- OpenGameArt.org
- Zapsplat.com (free tier)

Only use pure nature sounds - no music.
